Santunan Yatim

Program kepedulian berkelanjutan untuk mendukung kehidupan dan masa depan anak yatim piatu.

Beranda / Program / Santunan Yatim

Tentang Program

Program Santunan Yatim merupakan salah satu program unggulan RAGE SOKMA yang berfokus pada pemenuhan kebutuhan dasar, pendidikan, dan pendampingan anak yatim piatu secara berkelanjutan.

Program ini dilaksanakan melalui pendekatan sosial, keagamaan, dan kemanusiaan dengan melibatkan relawan, donatur, serta masyarakat sekitar.
