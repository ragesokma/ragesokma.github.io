Santunan Anak Yatim

Program kepedulian berkelanjutan untuk mendukung kehidupan dan masa depan anak yatim piatu.

Beranda / Program / Santunan Anak Yatim

Tentang Program

Program Santunan Anak Yatim merupakan salah satu program unggulan RAGE SOKMA yang berfokus pada pemenuhan kebutuhan dasar, pendidikan, dan pendampingan anak yatim piatu secara berkelanjutan.

Program ini dilaksanakan melalui pendekatan sosial, keagamaan, dan kemanusiaan dengan melibatkan relawan, donatur, serta masyarakat sekitar.

Santunan Rutin

Penyaluran bantuan secara rutin kepada anak yatim piatu.

Pendampingan Pendidikan

Dukungan pendidikan formal dan non-formal.

Pembinaan Keagamaan

Pembinaan akhlak dan spiritual melalui kegiatan keagamaan.

Kegiatan Sosial

Bakti sosial dan kegiatan kemanusiaan lainnya.
